> 通过在app.js中声明globalData，将所需要的变量及其值放在这里
1. 定义声明
```js
globalData: {
    userInfo: null,
    // dev or prod
    platform: "dev",
    dev_server: "xxxx",
    prod_server: "xxxxxx",
    feedback_server: "xxxxx"
  }
```
2. 引用
```js
// 在需要使用全局变量的地方，声明变量
const app = getApp();
// 引用
var paltform = app.globalData.platform
...
...
```
